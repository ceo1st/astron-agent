      <h2>典型落地方向</h2>
      <p>从信息理解到流程执行，Astron Agent 的落地价值主要集中在以下四类方向。</p>
    </div>
    <div class="astron-cases__pillars">
      <article class="astron-cases__pillar">
        <h3>智能分析与决策</h3>
        <p>面向非结构化数据清洗、市场分析和企业知识问答，让信息处理从人工整理转向智能提炼。</p>
      </article>
      <article class="astron-cases__pillar">
        <h3>Agent + RPA 协同执行</h3>
        <p>覆盖业务受理、订单处理、业务稽核、邮件和 Excel 自动化，让智能体真正完成执行闭环。</p>
      </article>
      <article class="astron-cases__pillar">
        <h3>私有化与内网适配</h3>
        <p>适配金融、电信、制造和涉密网络环境，兼顾高安全要求、复杂网络和存量系统集成。</p>
      </article>
      <article class="astron-cases__pillar">
        <h3>二次开发与共建</h3>
        <p>支持围绕 SSO、企微集成、插件扩展和 Skill 演进持续建设企业自己的智能体能力。</p>
      </article>
    </div>
