# 配置说明

本页收敛 Astron Agent 接入时最常见的配置关注点，帮助你快速判断应该优先看哪些配置文档。

## 重点配置域

### 基础设施

- 数据库与缓存
- 对象存储
- 网络与反向代理
- 容器运行环境

### 模型与 AI 能力

- 模型网关或 MaaS 接入参数
- API Key 与鉴权信息
- 能力开关与模型路由策略

### 平台能力

- MCP 和外部工具调用配置
- RPA 相关执行环境
- 多租户、空间和团队能力

### 鉴权与安全

- Casdoor 等身份系统接入
- 环境变量管理
- 生产环境密钥注入和权限边界

## 配置建议

1. 先完成最小可运行配置
2. 再补充生产环境依赖和密钥管理
3. 最后根据业务场景接入 MCP、RPA 和扩展模块
